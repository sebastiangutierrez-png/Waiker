# Waiker

A dashboard for farm operations and agronomic monitoring.

Structure:
- `src/` — HTML, CSS, and JS

Quick start:
1. Open `src/index.html` in a browser, or use a static server.
2. For local development, run a simple dev server (e.g., `npx http-server src` or `npx vite`).

This repo is a prototype UI and uses simulated data.
