Historial completo

Trazabilidad agrícola

La idea es que cada lote o árbol tenga una historia: siembra, mantenimiento, clima, fotos, trabajadores, insumos, cosecha y código final.

Asistente LIA demo

Hola. Soy una simulación del asistente LIA. Todavía no estoy conectado a OpenCloud, pero puedes probar preguntas como: “¿qué hago hoy?”, “estado del lote B” o “alertas”.

Qué hará este asistente después

1. Leer datos reales de la finca.
2. Revisar clima, humedad, fotos y tareas.
3. Generar recomendaciones diarias.
4. Enviar instrucciones por WhatsApp.
5. Explicar dashboards en lenguaje sencillo.
6. Pedir aprobación humana en decisiones delicadas.
